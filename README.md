# CarlosArocha.me

A lightweight GitHub Pages site inspired by the overall layout of alexlenail.me, implemented with original HTML and CSS.

## Deploy

1. Copy all files in this folder into the root of your local `arocha.github.io` repository.
2. Keep `CNAME` unchanged; it should contain `carlosarocha.me`.
3. Replace the lorem ipsum text in `index.html` with the final biography.
4. Commit and push:

```bash
git add .
git commit -m "Use Alex-inspired homepage layout"
git push origin main
```

GitHub Pages will redeploy automatically.
